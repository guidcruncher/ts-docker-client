🐳 docker-client

A robust, type-safe TypeScript ESM library for interacting with the Docker Engine API.

🚀 Features

  • Full Docker API coverage
  • Supports Unix, TCP, HTTPS, SSH, and Windows named pipe
  • TLS certs, proxy, and custom headers
  • Streaming logs, exec, and event listener
  • Config validation and environment fallback
  • Filtering, pagination, label discovery and diffing
  • Reactive label watcher and caching
  • Debug logging with optional custom logger

📦 Installation

npm install docker-client

🔌 Connectivity

configureDocker({
  transport: 'tcp',
  host: 'localhost',
  port: 2375
});

🌍 Environment Fallback

configFromEnv(); // Uses DOCKER_HOST, DOCKER_TLS_VERIFY, DOCKER_CERT_PATH

✅ Configuration Validation

validateDockerConfig(config);

🔧 API Examples

listContainers();
pullImage('nginx:latest');
createVolume('my-data');
getDockerInfo();
streamLogs('container-id');

🔍 Filtering

buildDockerFilters({ label: ['env=prod'], status: 'running' });

🏷️ Labels

extractContainerLabels(container);
collectUniqueLabels([container1, container2]);
diffLabels(oldLabels, newLabels);

🔁 Label Watcher

watchLabelChanges((id, labels) => {
  console.log(id, labels);
});

🧰 Caching

getCached('containers');
setCached('containers', containers);

🐞 Logging

enableDebugLogging();
setCustomLogger((level, msg) => myTelemetry.capture(level, msg));
